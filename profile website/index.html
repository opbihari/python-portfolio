🚀 Featured Mini-Projects

AI_Sage_Web_Project

Python Function Practice with AI

A robust Flask-based backend API that integrates with multiple LLM providers (Google Gemini, OpenAI GPT-4, Anthropic Claude). It exposes endpoints to automatically generate Python code snippets from natural language prompts, explain code logic, and suggest refactoring improvements.

Concepts Demonstrated:

RESTful API Design (Flask), LLM Integration, Environment Variable Management, JSON Parsing, Error Handling, Mock Services.

DDoS_Detector_Project

DDoS Detector & Firewall Simulator

A security-focused project that analyzes server logs to identify potential Distributed Denial of Service (DDoS) attacks. It processes raw text logs, counts IP requests, flags suspicious activity, and outputs the analysis to a CSV report. Includes scripts to generate sample logs and simulate a basic firewall response.

Concepts Demonstrated:

Cybersecurity Concepts (Rate Limiting), Log File Parsing, Data Sanitization, Dictionary Operations for Counting, CSV Generation.

IP_Blacklist_Checker_Project

IP Blacklist Checker

An interactive security tool that checks IP addresses against a simulated threat intelligence database (CSV). It provides real-time alerts based on threat levels and previous blocks, demonstrating how to build a basic command-line security scanner.

Concepts Demonstrated:

File I/O (CSV Parsing), Interactive CLI Loops, Threat Intelligence Simulation, Exception Handling.

Web_Scraping_Project

Web Scraping Data Extractor

A web scraper built using Python's requests and BeautifulSoup libraries. It connects to a target website, parses the HTML DOM to extract specific data elements, and saves the structured output into a CSV file. It also includes a search function to quickly retrieve scraped data.

Concepts Demonstrated:

Web Scraping (BeautifulSoup), HTTP Requests, HTML Parsing, DOM Traversal, CSV Data Storage.

Contact_Management_System.ipynb

Contact Management System

A fully functional CLI-based contact management system that allows users to add, view, and delete contacts. The application features persistent data storage utilizing CSV files, ensuring that data is saved correctly across sessions.

Concepts Demonstrated:

File I/O (CSV manipulation), Dictionary and List Comprehensions, CRUD Operations, State Persistence, CLI Menus.

Online_Book_Store_System.ipynb

Online Book Store Inventory System

An interactive bookstore inventory management application. It handles tasks such as adding new books, tracking unique genres, updating information for existing books, removing sold books, and maintaining an up-to-date inventory list.

Concepts Demonstrated:

Advanced List Operations, Tuple Unpacking, Data Validation, Error Handling, Set Operations (for unique items).
